# 🏙️ The Civic Accountability Pipeline (Work in Progress)

**Goal:** To build an automated data pipeline that scrapes, cleans, and visualizes municipal complaints to track how long it *actually* takes for civic issues to be resolved.

## 📌 Why I'm Building This
I got tired of seeing civic complaints disappear into a bureaucratic black hole. I am building a cloud-native data pipeline to analyze tens of thousands of real-world civic grievances to find the bottlenecks. 

**I am building this in public.** You can follow the daily progress, code updates, and SQL findings on my Twitter: https://x.com/Wings435104

## 🛠️ Planned Architecture
* **Extraction & Cleaning:** Python (Pandas) via Google Colab.
* **Database:** Cloud SQL (BigQuery/Supabase) for fast querying.
* **Visualization:** Streamlit & Plotly for interactive dashboards.
